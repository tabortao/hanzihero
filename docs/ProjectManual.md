
# 汉字小英雄 (Hanzi Hero) - 项目说明文档

## 1. 项目简介

**汉字小英雄** 是一款专为小学生设计的游戏化识字应用。它结合了先进的 AI 技术和科学的记忆方法，旨在让汉字学习变得生动有趣、高效且个性化。应用通过多种互动游戏模式，帮助孩子从“认字”到“用字”，最终实现独立阅读。

## 2. 核心设计理念

本项目基于以下核心教育理念设计：

### 2.1 3-1-3 识字法
这是本应用“每日挑战”功能的基础逻辑，旨在平衡新知识的学习与旧知识的巩固：
*   **3 (新学)**：每天学习 3 个（或更多，可设置）新的生字。
*   **1 (短期回顾)**：复习 1 天前（昨天）学习过的汉字，巩固短期记忆。
*   **3 (长期回顾)**：复习 3 天前学习过的汉字，防止遗忘。

### 2.2 艾宾浩斯遗忘曲线 (Ebbinghaus Forgetting Curve)
应用内置了智能复习算法，根据艾宾浩斯记忆原理，自动追踪每个汉字的学习时间。系统会在关键的遗忘节点（如 1天、3天、7天、15天、30天）将汉字重新推送到“每日挑战”中，帮助孩子将短期记忆转化为长期记忆。

### 2.3 AI 驱动 (AI-Powered)
*   **智能讲解**：AI 老师（基于 Google Gemini 或 OpenAI 兼容模型）能实时分析汉字，提供生动的字形结构拆解、趣味顺口溜、组词和造句。
*   **短文生成**：应用能根据孩子“我的字库”中已掌握的汉字，生成独一无二的趣味短文。这实现了“识字”到“阅读”的无缝衔接，让孩子在阅读中获得成就感。
*   **游戏内容生成**：游戏关卡的内容（如“汉字消消乐”的词语配对）由 AI 实时生成，确保证题目新颖且具有针对性。

## 3. 主要功能模块

### 3.1 识字 (Home)
*   **教材同步**：内置人教版、苏教版等主流教材，支持按年级、单元进行系统学习。
*   **自定义单元**：支持家长或老师创建自定义学习单元，输入汉字即可自动生成拼音和课程。
*   **学习卡片**：提供汉字笔画动画、田字格展示、拼音发音、AI 辅助讲解（结构、记忆法、组词、造句）。

### 3.2 游戏化学习
应用包含多种寓教于乐的游戏模式：
*   **智趣翻卡屋 (Classic)**：经典的翻转卡片学习模式，专注于认读和记忆。
*   **星箭追击 (Listen)**：听音辨字游戏。孩子化身弓箭手，根据听到的读音射击正确的汉字，锻炼听辨能力。
*   **超能靶场 (Look)**：看拼音识字游戏。根据显示的拼音，在气泡中找到对应的汉字，锻炼拼读能力。
*   **汉字消消乐 (Crush)**：三消类游戏。将汉字与对应的拼音/词语进行配对消除，AI 自动生成关卡，复习生字本中的难点字。

### 3.3 阅读 (Story)
*   **AI 故事创作**：输入关键词，AI 会利用孩子已认识的汉字编写故事。
*   **分级阅读**：生成的故事难度自动适配孩子的识字量。
*   **互动阅读**：故事中的生字可点击发音、加入生字本，实现沉浸式学习。

### 3.4 统计与复习 (Stats)
*   **学习数据**：直观展示已识汉字量、生字本数量。
*   **热力图**：记录每日学习活跃度，鼓励坚持。
*   **记忆曲线**：可视化展示当前汉字的记忆状态（新学、1天、3天、7天等）。
*   **生字本**：自动收集学习中标记为“不认识”或游戏出错的汉字，支持针对性复习。

### 3.5 个人中心 (Profile)
*   **设置**：配置 AI 模型（API Key）、每日学习量、语音朗读速度等。
*   **数据管理**：支持学习进度的导出与导入（备份），防止数据丢失。

## 4. 使用说明

1.  **初始化**：首次进入应用，建议在“我的”页面配置教材版本和年级，并设置 AI API Key（支持 Google Gemini 或 OpenAI 兼容接口）。
2.  **开始学习**：在首页选择一个单元，点击“开始学习”进入卡片学习模式。
3.  **每日挑战**：点击首页左侧的“每日挑战”，系统会自动筛选需要复习和新学的汉字，生成今日任务。
4.  **巩固练习**：通过“星箭追击”、“超能靶场”等游戏巩固今日所学。
5.  **阅读拓展**：学习一定数量汉字后，进入“阅读”页面，生成专属故事进行阅读训练。
